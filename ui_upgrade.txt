Integrating Tournify Redesign into Next.js

To move your Next.js project over to this new "Organic Athletics" UI, follow these steps:

1. Update tailwind.config.js

Add the new color palette and font to your configuration. This ensures the custom Matcha and Peach colors are available as utility classes.

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./pages/**/*.{js,ts,jsx,tsx,mdx}",
    "./components/**/*.{js,ts,jsx,tsx,mdx}",
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {
      colors: {
        matcha: {
          DEFAULT: '#8A9A5B',
          light: '#A4B475',
          dark: '#6E7B49',
          surface: '#F5F4EB',
        },
        peach: {
          DEFAULT: '#FFCBA4',
          light: '#FFE4D1',
          dark: '#EBB58F',
        },
        cream: {
          DEFAULT: '#FAFAFA',
          soft: '#FBF9F1',
        }
      },
      fontFamily: {
        sans: ['Plus Jakarta Sans', 'sans-serif'],
      },
    },
  },
  plugins: [],
}

2. Update globals.css

Ensure you're importing the correct font and setting the base background color.

@import url('https://fonts.googleapis.com/css2?family=Plus+Jakarta+SANS:wght@400;500;600;700;800&display=swap');

@tailwind base;
@tailwind components;
@tailwind utilities;

body {
  background-color: #FAFAFA;
  color: #2D2D2D;
  font-family: 'Plus Jakarta Sans', sans-serif;
}
